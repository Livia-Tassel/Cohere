## 🚀 下一步：部署到线上

你现在可以开始部署了！我推荐使用 **Vercel + Render** 组合。

### 快速部署步骤（15 分钟）

#### 第 1 步：部署后端到 Render（5 分钟）

1. **访问** https://render.com
2. **登录** 使用 GitHub 账号
3. **创建 Web Service**
   - 选择 `Cohere` 仓库
   - Root Directory: `server`
   - Build Command: `npm install`
   - Start Command: `npm start`

4. **添加环境变量**：
   ```
   PORT=5000
   MONGODB_URI=mongodb+srv://livia:20041014@cohere.i32mvls.mongodb.net/?appName=Cohere
   JWT_SECRET=34cfbdc028549964a4c94c6a7daf65ad315da32e95d5aa0dc0c04ea7a7a86f70
   NODE_ENV=production
   ```

5. **部署并获取 URL**
   - 例如：`https://cohere-api.onrender.com`

#### 第 2 步：部署前端到 Vercel（5 分钟）

1. **访问** https://vercel.com
2. **登录** 使用 GitHub 账号
3. **导入项目**
   - 选择 `Cohere` 仓库
   - Framework: Vite
   - Root Directory: `client`
   - Build Command: `npm run build`
   - Output Directory: `dist`

4. **添加环境变量**：
   ```
   VITE_API_URL=https://cohere-api.onrender.com/api
   ```
   （使用你的 Render URL）

5. **部署并获取 URL**
   - 例如：`https://cohere.vercel.app`

#### 第 3 步：更新 CORS（2 分钟）

1. **回到 Render**
2. **添加环境变量**：
   ```
   CORS_ORIGIN=https://cohere.vercel.app
   ```
   （使用你的 Vercel URL）

3. **等待重新部署**（2-3 分钟）

## 🔐 安全提醒

### ⚠️ 重要：保护你的密钥

当前配置中包含敏感信息，建议：

1. **更改 MongoDB 密码**（推荐）
   - 登录 https://cloud.mongodb.com
   - Database Access → 修改密码
   - 更新部署平台的环境变量

2. **生成新的 JWT_SECRET**（可选）
   ```bash
   node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
   ```

3. **限制数据库访问**（可选）
   - Network Access → 添加特定 IP
   - 或保持 0.0.0.0/0（允许所有）
